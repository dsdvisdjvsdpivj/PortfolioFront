ParkyApi & ParkyMVC

ParkyAPI & ParkyMVC zijn 2 webapplicaties die ik heb ontwikkeld tijdens het volgen van een udemy course van Bhrugen Patel. Tijdens deze course heb ik 2 projecten gebouwd. ParkyAPI is een REST Api die informatie bevat over verschillende nationale parken. Deze Api word vervolgens uitgelezen door ParkyMVC. ParkyMVC is een ASP.Net Core 3 MVC Project. dit project maakt gebruik van de HttpClient functionaliteiten om de data van de REST api op te vragen.

Uplift

Uplift is een webapplicatie ontwikkeld in ASP.Net Core 3. Deze applicatie is ontwikkeld tijdens het volgen van een udemy course van Bhrugen Patel. Via deze applicatie kan een klant een afspraak bij een lokale dienstverlener. Waar tijdens het Spice project nog voornamelijk de focus lag op het verwerken van data via CRUD operaties, Heb ik tijdens het Uplift Project ervaring opgedaan met Repository Patterns. Ook is de code verdeeld over meerdere layers, door de code op deze manier te verdelen is er een strikte scheiding tussen de verschillende functies van de applicatie

Spice

Spice is een webapplicatie ontwikkeld in ASP.Net Core 3. Deze applicatie is ontwikkeld tijdens het volgen van een udemy course van Bhrugen Patel. Via deze applicatie kan een klant een bestelling plaatsen bij een fictief restaurant. vervolgens kan de klant een fictieve betaling uitvoeren via de stripe api. Ook kunnen er via de sendgrid api mails worden verstuurd over de status van de order. Voor de medewerkers van het fictieve restaurant is een Admin sectie toegevoegd.
